# 🏢 Apartment Search Bot — Dushanbe Real Estate Assistant

A simple, fast, and user-friendly Telegram bot for helping users search for apartments in **Dushanbe, Tajikistan**. Built for a real estate company that owns multiple buildings across the city, the bot allows users to search for available flats by district, while providing an admin interface to manage listings.

## ✨ Features

- 🔍 **User-friendly search**  
  Users can easily search for available apartments using 4 main districts of Dushanbe.

- 🏙️ **District filters**  
  Search by:
  - Sino
  - Firdavsi
  - Shohmansur
  - Ismoili Somoni

- 👤 **Admin panel**  
  - Add or update apartment listings  
  - Manage housing inventory  

- 📦 **Persistent storage**  
  Apartment data is stored and retrievable across sessions.

## ⚙️ Technologies Used

- Python 🐍  
- aiogram (Telegram bot framework)  
- JSON or database for storage  
- python-dotenv for managing environment variables
